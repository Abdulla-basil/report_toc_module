Report Table of Contents (ToC)

Transform long, complex PDF reports into professional documents with automated navigation and dynamic page mapping.

Why Use This Module?

✔ Dynamic Mapping

Scans your PDF content to find specific headings and maps them to accurate page numbers.

✔ Seamless Insertion

Inject your ToC at any specific page index without breaking document flow.

✔ Auto-Pagination

Automatically replaces and updates footers to ensure page numbers account for new ToC pages.

⚠️ Technical Requirement: This module uses the PyMuPDF (fitz) library. Ensure pip install pymupdf is executed on your server environment.

Centralized Configuration

Easily manage all your Report ToCs under the Technical menu.

Comprehensive Rule Engine

Configure page insertion, PDF templates, and search markers in one view.

Define your Search Text (right) to find content in the PDF and your ToC Heading (left) to display it in the Table of Contents.

Professional Output

Beautifully rendered ToC pages that match Odoo 18's design language.
